Task-04: Sentiment Analysis on Social Media Data

This task analyzes sentiment patterns in social media text data to understand public opinion and attitudes toward different entities.

Dataset-
Source: Kaggle-https://www.kaggle.com/datasets/jp797498e/twitter-entity-sentiment-analysis

Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

Key Insights:
- Positive sentiment is dominant in the social media dataset.
- Negative sentiment highlights user dissatisfaction toward certain entities.
- Neutral sentiment represents informational or opinion-free content.
- Social media sentiment analysis helps understand public opinion effectively.
